初识 JAVA

JAVA 的诞生

  1. 1972 年 C 诞生

    • 贴近硬件，效率高
    • 操作系统，编译器，数据库，网络系统等
    • 指针和内存管理

  2. 1982 年 C++

    • 面向对象
    • 兼容 C
    • 图形领域、游戏等

  3. 建立新语言 JAVA

    • 语法相似 C
    • 没有指针
    • 没有内存管理
    • 真正的可移植性
    • 面向对象
    • 类型安全
    • 高质量类库
    • ……

    1985 年

    1. 图形界面程序（Applet）
    2. J2SE 占领桌面（界面不美观，需要虚拟机）
    3. J2ME 占领手机
    4. J2EE 占领服务器

    JAVA 三高：高可用、高性能、高并发。

JAVA 三大版本

Write once, Run anywhere.(JVM)

  1. JavaSE:标准版（桌面程序，控制台开发）
  2. JavaME:嵌入式开发
  3. JavaEE：企业级开发（web 端，服务器开发）

JDK、JRE、JVM

  • Java Development Kit
  • Java Runtime Environment
  • Java Virtual Machine

JAVA 程序运行机制

区别于其执行的时机

  • 编译型（complie，执行速度快）
  • 解释型（explain，开发对速度要求不高的网页等）

JAVA 数据类型（强类型语言）

  1. 要求变量的使用要严格符合规定，先定义后使用

    • 安全性高但速度慢

  2. Java 数据类型分为两大类

    1. 基本类型（primitive type）

      1. 数值类型

        1. 整数类型

          1. byte 占 1 个字节范围：-128-127

          2. short 占 2 个字节范围：-32768-32767

          3. int 占 4 个字节范围:-2147483548-2147483547

          4. long 占 8 个字节范围：-9223372036854775808-9223372036854775807

            • long 定义时在数字后+L

        2. 浮点类型

          1. float 占 4 个字节范围

            • long 定义时在数字后+L

          2. double 占 8 个字节范围

        3. 字符类型

          1. char 占 2 个字节范围

      2. boolean 类型（占 1 位，1bit，1B = 8Bit）

    2. 引用类型（reference type）

      1. 类
      2. 接口
      3. 数组
